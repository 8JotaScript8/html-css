Codigo-fonte e pré-formatação

Para representar codigos podemos usar a TAG <code>
ex. document.getElementById('x')
Para criar um campo para manipular identações e quebras de linhas em uma representação de codigo usamos o <pre>
ex. 

            num = int(input('Digite um número: '))
            
            if num % 2 == 0:
                print(f'{num} é par')
            else:
                print(f'{num} é impar')
