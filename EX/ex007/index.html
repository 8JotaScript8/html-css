Principais formatações

OBS: de preferencia as tags semanticas

Negrito:

Podemos usar a TAG <b>(não semantica); ou a TAG strong(semantica)

Itálico:

Para um termo em Itálico podemos usar a TAG <i>(não semantica) ou a TAG <em>(semantica)

Marca texto:

Podemos marcar um texto usando a TAG <mark>

Texto grande e pequeno:

Para um texto grande usamos o <big>(tag obsoleta) e <small> para um pequeno

Texto riscado:

Para riscarmos um texto usamos a TAG <del>

Texto sobrescrito:

EX. x2+3. Usamos a TAG <sup>

Texto subscrito:

EX. H2O. Usamos a TAG <sub<
